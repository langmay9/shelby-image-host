# Shelby Image Host Demo

This is a simple demo project showing how Shelby decentralized storage can host and deliver high resolution images.

## Features

- Fast loading gaming images
- Simple image gallery
- Shelby decentralized storage concept

## Demo Use Case

Gaming communities often need fast and reliable media hosting for wallpapers and game clips.

Shelby can be used to store and deliver these assets instantly.
